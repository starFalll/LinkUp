# LinkUp Mini Program

原生微信小程序 + 微信云开发的约饭 MVP。

## 项目结构

- `miniprogram/`: 小程序前端代码
- `cloudfunctions/`: 云函数
- `docs/cloud-db.md`: 数据表和云开发初始化说明

## 快速开始

1. 用微信开发者工具导入当前目录。
2. 如果只想先看界面和流程，不配置云环境也可以直接运行。
3. 如果要启用真实云开发：
   - 在微信开发者工具中创建云环境。
   - 把 `miniprogram/config/runtime.js` 中的 `cloudEnvId` 改成你的环境 ID。
   - 在云开发数据库中创建文档里列出的集合。
   - 上传 `cloudfunctions/` 下的全部云函数。
   - 建议给 `refreshExpiredPosts` 配一个定时触发器，例如每 10 分钟运行一次。

## 功能说明

- 首页双列瀑布流浏览约饭帖子
- 发帖时选择楼宇、时间、内容、位置、剩余座位和是否需申请
- 直接加入或申请加入
- 发起人在“我的发起”中审批申请
- 我的历史记录、常约饭搭子榜、全站活跃榜

## 说明

- 当前仓库包含一套本地 mock 数据逻辑，方便在没有云环境时预览完整交互。
- 所有会影响座位数的操作在真实云开发模式下都通过云函数处理。
- 列表和详情在真实云环境中也通过云函数读取，避免“需申请帖子”提前暴露具体位置。
